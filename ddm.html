BIOPHYSICS DDM MICROSCOPY PYTHON

Multimodal Transport via
Differential Dynamic Microscopy

Quantifying coexisting transport modes without tracking a single particle.

What is DDM?

Differential Dynamic Microscopy is a Fourier-space technique that extracts dynamic information from a sequence of microscopy images — without ever needing to locate, identify, or track individual particles. Instead of asking "where is this particle now, and where was it before," DDM asks "how much does the image change, on average, over a given time lag and spatial frequency?"

The result is an image structure function that, when fit to appropriate models, yields transport parameters — diffusion coefficients, drift velocities, fractions of each population — directly from the raw image data.

The Problem

Many physical and biological systems don't have the decency to exhibit a single clean transport mode. Particles diffuse. Motors walk. Some do both. Some do neither cleanly. Conventional tracking methods struggle when populations overlap, when particles are too dense to resolve, or when the motion is too fast or too slow for the frame rate.

The question this project addressed: can DDM reliably decompose a system exhibiting multiple simultaneous transport modes — diffusion and directed motion coexisting in the same field of view — and return accurate parameters for each?

Approach

Built a Python pipeline to compute the image structure function from raw video data, fit multimodal models to the resulting decay curves across spatial frequencies, and extract transport parameters for each population. Validated against synthetic data with known ground truth before applying to experimental systems.
